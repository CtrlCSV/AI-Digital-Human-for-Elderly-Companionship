# c4proj 接口文档

本文档描述 `demo/server.py` 与 `demo/public/script.js` 当前统一后的接口契约。

## 统一约定

- 角色标识统一使用：`girl`、`boy`、`custom`。
- 头像 ID 约定：`1=girl`，`3=boy`，`99=custom`。
- 预设头像静态文件统一使用前端现有文件名：
  - `1` -> `/static/avatar-xiaoli.<ext>`
  - `3` -> `/static/avatar-xiaoming.<ext>`
- 自定义头像统一使用 `/static/custom_avatars/custom_<hash>.<ext>`。
- WebSocket 消息均为 JSON，必须包含 `type` 字段。

## REST 接口

### `GET /`

返回前端页面 `public/index.html`。

### `GET /api/weather?city=<城市>`

查询天气。实际字段由 `tools.get_weather()` 返回结构决定。

### `GET /api/news?topic=<主题>`

查询新闻。实际字段由 `tools.get_news()` 返回结构决定。

### `GET /api/calendar`

查询日期日历信息。实际字段由 `tools.get_calendar()` 返回结构决定。

### `POST /asr`

语音识别接口。请求体为音频二进制数据。

### `POST /api/avatar/upload`

上传头像图片。

请求类型：`multipart/form-data`

| 字段 | 类型 | 必填 | 说明 |
|---|---|---:|---|
| `file` | file | 是 | 支持 `image/png`、`image/jpeg`、`image/jpg`、`image/webp`，最大 10MB |
| `slot` | int | 否 | `1/3` 表示替换预设头像，其他值表示自定义头像 |

响应示例：

```json
{
  "status": "ok",
  "url": "/static/custom_avatars/custom_d1a804b4.jpg",
  "slot": null,
  "filename": "custom_d1a804b4.jpg"
}
```

预设头像上传响应中的 `slot` 为 `1/3`，`url` 为对应 `/static/avatar-xiaoli.<ext>`、`/static/avatar-xiaoming.<ext>`。

### `GET /api/dialects`

获取方言配置。

```json
{
  "dialects": [
    { "id": "mandarin", "label": "普通话" },
    { "id": "cantonese", "label": "粤语" },
    { "id": "taiwanese", "label": "台湾腔" }
  ]
}
```

### `GET /api/idle-video/playlist?role=<角色>`

获取空闲视频播放列表。

| 参数 | 可选值 |
|---|---|
| `role` | `girl`、`boy`、`custom` |

响应示例：

```json
{
  "urls": ["/static/idle_boy_0.mp4"]
}
```

## WebSocket 接口

连接地址：`/ws/chat`

### 前端发送消息

#### `init`

```json
{
  "type": "init",
  "avatarId": 1,
  "avatarName": "小丽",
  "sessionId": "default",
  "userName": "张三",
  "userId": "10001",
  "dialect": "mandarin",
  "city": "上海"
}
```

#### `new_session`

```json
{
  "type": "new_session",
  "sessionId": "1710000000000",
  "userName": "张三",
  "userId": "10001"
}
```

#### `switch_session`

```json
{
  "type": "switch_session",
  "sessionId": "1710000000000",
  "userName": "张三",
  "userId": "10001"
}
```

#### `message`

```json
{
  "type": "message",
  "content": "今天天气怎么样？",
  "sessionId": "1710000000000",
  "timestamp": 1710000000000,
  "userName": "张三",
  "userId": "10001",
  "city": "上海"
}
```

兼容类型：`text`、`barge_in_commit`。文本字段兼容 `content` 或 `data`。

#### `audio`

```json
{
  "type": "audio",
  "data": "data:audio/wav;base64,..."
}
```

`data` 也兼容 PCM 数组。

#### `frame`

```json
{
  "type": "frame",
  "data": "data:image/jpeg;base64,..."
}
```

#### `set_dialect`

```json
{
  "type": "set_dialect",
  "dialect": "cantonese"
}
```

#### `barge_in_start`

```json
{
  "type": "barge_in_start",
  "timestamp": 1710000000000
}
```

#### `delete_user`

```json
{
  "type": "delete_user",
  "userId": "10001"
}
```

#### `location`

```json
{
  "type": "location",
  "city": "上海"
}
```

#### `ping`

```json
{
  "type": "ping"
}
```

### 后端发送消息

#### `pong`

```json
{
  "type": "pong"
}
```

#### `turn_start`

```json
{
  "type": "turn_start",
  "responseId": 12,
  "userText": "你好"
}
```

#### `assistant_text_delta`

```json
{
  "type": "assistant_text_delta",
  "responseId": 12,
  "delta": "你好呀，我在这里。",
  "fullText": "你好呀，我在这里。"
}
```

#### `assistant_chunk`

```json
{
  "type": "assistant_chunk",
  "responseId": 12,
  "seq": 0,
  "text": "你好呀，我在这里。",
  "fullText": "你好呀，我在这里。",
  "isFinalChunk": true,
  "video": "base64..."
}
```

字段说明：

| 字段 | 说明 |
|---|---|
| `video` | MP4 base64，有视频时优先返回 |
| `audio` | 音频 base64，无视频但有语音时返回 |
| `fallback` | `text_only`，表示仅文本 |

#### `turn_end`

